# ckb-nft-toolkit

### Quick Start

Rename `.env.example` to `.env` and update the env variables

```
yarn install
```

Update the [example](https://github.com/duanyytop/ckb-nft-toolkit/tree/develop/src/example), select the function you want.

```
yarn start:issuer

yarn start:class

yarn start:nft
```
